# Requirements

## Prerequisites
- Node V12.18.0

## Frameworks
- Truffle
- ReactJs
- NodeJs

## Setup
- Create a new directory and initialize npm with ```npm init -y```
- Add Truffle dependency with ```npm install --save truffle@5.1.8```
- Add Web3Js dependency with ```npm install --save web3```
